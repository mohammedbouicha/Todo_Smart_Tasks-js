# Todo_Smart_Tasks-js
Todo Smart est une application web simple et développée en HTML, CSS et JavaScript, permettant de gérer efficacement leurs tâches quotidiennes. 
---

## Fonctionnalités
- Ajouter une tâche
- Modifier une tâche
- Marquer une tâche comme terminée
- Supprimer une tâche
- Supprimer toutes les tâches avec message de confirmation
- Filtrer les tâches (toutes / actives / terminées)
- Sauvegarde automatique avec `localStorage`
- identifiants uniques (`Date.now()`)

---
## Technologies utilisées
- HTML5
- CSS3
- JavaScript
